# Valentine's Day App - Frontend

A romantic Valentine's Day interactive web application built on the Internet Computer.

## Features

- Interactive Valentine's Day proposal with playful UI
- Responsive design for mobile and desktop
- Romantic pink and white color theme
- Custom domain support for personalized URLs

## Custom Domain Setup

This application is hosted on the Internet Computer and can be accessed via a custom domain. To set up your own domain:

1. **Access the Custom Domain Panel**: Click the settings icon (⚙️) in the top-right corner of the app
2. **Enter Your Domain**: Input your desired domain name (e.g., `myvalentine.com` or `valentine.mysite.com`)
3. **Configure DNS**: Follow the on-screen instructions to add the required DNS records at your domain registrar

For detailed instructions, troubleshooting, and best practices, see the [Custom Domain Documentation](./docs/custom-domain.md).
